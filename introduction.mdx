# Welcome to Rigbox

Rigbox provides secure, isolated Firecracker micro-VM workspaces for running AI agents, web services, and development environments. Each workspace is a full Linux VM with SSH access, app routing, snapshots, tools, and live metrics.

## Base URL

All API requests are made to:

```bash
https://api.rigbox.dev/api
```

## Authentication

Most endpoints require authentication via API key or Bearer token. Public endpoints include `/capacity` and `/templates`.

```bash
curl https://api.rigbox.dev/api/workspaces \
  -H "Authorization: Bearer YOUR_API_KEY"
```

<Note>
  Get your API key from the [Rigbox Dashboard](https://rigbox.dev) under **Settings → API Keys**.
</Note>

## Rate Limits

The API is rate-limited per user. If you receive `429 Too Many Requests`, retry with backoff.

## Response Format

All responses are JSON. Successful responses return `2xx` status codes. Errors return appropriate `4xx`/`5xx` codes with a message:

```json
{
  "message": "Workspace not found"
}
```
